Supported datatypes

The following datatypes are supported in result row columns.

  • Null
  • TinyInt
  • SmallInt
  • Int
  • Bit
  • DateTime
  • Numeric
  • VarChar
  • Char
  • NVarChar
  • NChar

Any other types are likely to fail until support is added for them.

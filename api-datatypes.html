Supported datatypes

The following datatypes are supported in result sets.

  • Null
  • TinyInt, SmallInt, Int, BigInt
  • Bit
  • SmallDateTime, DateTime
  • Numeric, Decimal
  • Real, Float
  • SmallMoney, Money
  • Char, VarChar, Varchar(max)
  • NChar, NVarChar, NVarChar(max)
  • Binary, VarBinary, VarBinary(max)
  • Text, NText, Image
  • UniqueIdentifier

BigInt

Values are returned as a string. This is because values can exceed 53 bits of significant data, which is great than a Javascript Number type can represent as an integer.

UniqueIdentifier

Values are returned as a 16 byte array of integer values.

Note that the order of bytes is not the same as the character representation. See Using uniqueidentifier Data for an example of the different ordering of bytes.

Other datatypes

Any other types are likely to fail until support is added for them.
